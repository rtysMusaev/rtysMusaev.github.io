
# Инструкция для работы с Markdown

## Выделение текста

Чтобы выделить текст курсивом необходимо обрамить его звездочками (*) или знаком нижнего подчеркивания (_). Например, *вот так* или _вот так_.

Чтобы добавить нумерованные списки, необходимо пункты выделить звездочкой (*) или знаком +. Например вот так:
* Элемент 1
* Элемент 2
* Элемент 3
* Элемент 4

Чтобы добавить нумерованные списки, необходимо пункты просто пронумеровать. Например вот так:
1. Первый пункт
2. Второй пункт
3. Третий пункт

## Работа с изображениями

Чтобы вставить изображение в текст, достаточно написать следующее: ![Сообщение](расположение файла) ![Привет, это тефтелька!](cat.jpg)
